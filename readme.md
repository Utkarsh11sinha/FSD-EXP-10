# Experiment 06: Blog Platform with Comments and User Profiles

## Aim

Build a blog platform with authentication, user profiles, posts, and comments.

## Included Files

- `index.html`, `styles.css`, `app.js`: interactive blog platform demo

## Features

- Sign up and sign in with local session state
- Create, edit, and delete blog posts
- Add comments under posts
- User profile cards and relational post ownership

## Note

The deployed demo uses browser storage so the experience remains fully interactive on Netlify.
